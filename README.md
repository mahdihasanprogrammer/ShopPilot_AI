# ShopPilot AI — AI-Powered E-Commerce Shopping Assistant

ShopPilot AI is a modern e-commerce application powered by Claude Agentic AI, delivering personalized smart recommendations and site-wide conversational shopping assistance.

## Project Structure
- `client/`: Next.js (App Router) + TypeScript + Tailwind CSS (v4) frontend.
- `server/`: Express.js + TypeScript + native MongoDB driver backend.
- `PRD.md`: Complete Product Requirement Document.
- `prompts.md`: 18-step sequential build instructions.

## Development Setup

### Client
1. Navigate to the client folder:
   ```bash
   cd client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it.
